Port
==================================================

This class is the blueprint for the port instances in the
:mod:`hub.port` module. Those instances are automatically instantiated on
boot, and further populated when devices are plugged in. You cannot import or
instantiate this class manually.

.. class:: Port

    Provides access to port configuration and devices on a port. Some methods
    and attributes can only be used if the port is in the right mode, as shown
    below.
